<div class="content">
    <h1>Documentatie voor Web Apps</h1>
    <p style="padding-bottom: 0px; max-width: 1050px;">Met Azure Web Apps kunt u websites bouwen en hosten in de programmeertaal van uw keuze, zonder hiervoor infrastructuur te beheren. Het biedt automatisch schalen en een hoge beschikbaarheid, ondersteuning voor zowel Windows als Linux (preview-versie) en maakt automatische implementaties mogelijk vanuit GitHub, Team Services of een willekeurige Git-repo. Ontdek met onze snelstartgidsen, zelfstudies en voorbeelden hoe u Azure Web Apps gebruikt.</p>
