# <a name="what-is-the-azure-database-migration-service-preview"></a>Wat is Azure Database migratie Service Preview?
De Azure-Service voor het migreren van Database is een volledig beheerde service die is ontworpen om in te schakelen naadloze migraties uit meerdere databasebronnen naar Azure Data platforms met minimale downtime. De service is momenteel in de openbare Preview met ontwikkelingsinspanningen is gericht op:

- Betrouwbaarheid en prestaties.
- Iteratieve toevoeging van de bron-doel-paren.
- Blijvende investeringen in wrijving gratis migraties.

## <a name="use-familiar-tools"></a>Gebruik bekende programma's
Migratie van Azure databaseservice integreert enkele functies van onze bestaande hulpprogramma's en services.  Dit biedt klanten met een uitgebreide, maximaal beschikbare oplossing. Maakt gebruik van Azure migratie databaseservice [gegevens migratie-assistent](http://aka.ms/dma) voor van beoordelingsrapporten.  Deze aanbevelingen helpen u bij de wijzigingen die vereist zijn voordat u een migratie uitvoert. Het is aan u eventuele vereiste herstel moet worden uitgevoerd. Wanneer u klaar bent voor het migratieproces begint, voert Azure Database migratie-Service de bijbehorende stappen. U kunt starten en vergeet uw migratie-projecten met de zekerheid van weten dat het wordt gebruikgemaakt van best practices bepaald door Microsoft.
