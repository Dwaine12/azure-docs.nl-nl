# <a name="what-is-the-azure-database-migration-service"></a>Wat is de Service Azure Database migreren?
De Azure-Service voor het migreren van Database is een volledig beheerde service die is ontworpen om in te schakelen naadloze migraties uit meerdere databasebronnen naar Azure Data platforms met minimale downtime.

## <a name="use-familiar-tools"></a>Gebruik bekende programma's
De migratie van Azure databaseservice integreert enkele van de functionaliteit van onze bestaande hulpprogramma's en services. Dit biedt klanten met een uitgebreide, maximaal beschikbare oplossing. De service wordt gebruikt de [gegevens migratie-assistent](http://aka.ms/dma) naar assessment rapporten genereren waarin aanbevelingen om u te begeleiden de wijzigingen die zijn vereist voordat u een migratie uitvoert. Het is aan u eventuele vereiste herstel moet worden uitgevoerd. Wanneer u klaar bent voor het migratieproces begint, voert de migratie van Azure databaseservice alle stappen van de bijbehorende. U kunt starten en vergeet uw migratie-projecten met zekerheid er rekening mee dat bekend is dat het wordt gebruikgemaakt van best practices, zoals wordt bepaald door Microsoft.

## <a name="regional-availability"></a>Regionale beschikbaarheid
