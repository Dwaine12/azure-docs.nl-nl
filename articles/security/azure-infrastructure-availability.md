# <a name="availability-of-azure-infrastructure"></a>Beschikbaarheid van de Azure-infrastructuur
Azure biedt een robuuste beschikbaarheid op basis van uitgebreide redundantie verkregen met virtualisatietechnologie. Azure biedt verschillende niveaus van redundantie voor maximale beschikbaarheid van gegevens van klanten.

## <a name="temporary-outages-and-natural-disaster"></a>Tijdelijke storingen en natuurramp
Het team van Microsoft Cloud-infrastructuur en bewerkingen ontwerpt, bouwt, werkt en beveiligt de cloudinfrastructuur. Dit team zorgt ervoor dat de Azure-infrastructuur van hoge beschikbaarheid en betrouwbaarheid, hoge efficiëntie, slimme schaalbaarheid en een veilige, persoonlijke & vertrouwde cloud leveren.

Zorg ervoor dat elektriciteit continue als er een korte onderbreking van de power optreedt blijft noodvoeding en enorme banken batterijen.

Genereren van noodgevallen processorvermogen back-up voor uitgebreide onderbrekingen en gepland onderhoud. Het datacenter wordt beheerd met beveiligingspersoneel brandstof reserves als een natuurlijke ramp optreedt.

Snelle en robuuste fiber glasvezelkabels netwerken verbinding datacenters met andere belangrijke hubs en internetgebruikers. Rekenuren voor algemene service tolerantie knooppunten dichter-belastingen op host voor de eindgebruikers kunnen de latentie te verminderen, geografische redundantie bieden en verbeteren. Een team van engineers werkt servicepakket services blijft beschikbaar voor klanten zijn.

Microsoft zorgt voor hoge beschikbaarheid via geavanceerde bewaking en respons op incidenten, service-ondersteuning, en maak een back-up van failoverfunctionaliteit. Geografisch verspreide datacenters van Microsoft operations werken 24/7/365. Het Azure-netwerk is een van de grootste waarde in de hele wereld. De distributie van inhoud en glasvezelkabels fiber netwerk verbindt datacenters en Edge-knooppunten, zodat de hoge prestaties en betrouwbaarheid.

## <a name="disaster-recovery"></a>Herstel na noodgeval
Azure houdt klantgegevens duurzame op twee locaties met de klant de mogelijkheid om de locatie van de back-site hebben. Op beide locaties onderhoudt Azure voortdurend meerdere (3) in orde replica's van de gegevens van de klant.

## <a name="database-availability"></a>Beschikbaarheid van de database
Azure zorgt ervoor dat een database Internet toegankelijk zijn via een Internet-gateway met database continue beschikbaarheid. Evalueert de gezondheid en status van de actieve databases bewaking met tussenpozen van 5 minuten.

## <a name="storage-availability"></a>Opslagbeschikbaarheid
Azure biedt opslag via een uiterst schaalbare en duurzame storage-service, waarmee de connectiviteit eindpunten zodat deze kan worden rechtstreeks door een consumerende toepassing toegankelijk. Via de storage-service worden binnenkomende aanvragen voor opslag efficiënt met transactionele integriteit verwerkt.
