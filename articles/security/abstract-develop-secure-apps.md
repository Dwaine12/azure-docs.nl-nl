# <a name="develop-secure-applications-on-azure"></a>Ontwikkelen van veilige toepassingen op Azure
## <a name="abstract"></a>Samenvatting
In dit artikel is een algemene handleiding voor de beveiligingsvragen en -besturingselementen die u in elke fase van de software development lifecycle overwegen moet bij het ontwikkelen van toepassingen voor de cloud. Implementatie van deze concepten voordat u uw product een release kunt u bouw software op beter te beveiligen. De aanbevelingen die zijn gepresenteerd in dit document zijn afkomstig van onze ervaring met Azure-beveiliging en de ervaringen van onze klanten.

Dit document is bedoeld als een resource voor softwareontwerpers van, ontwikkelaars en testers op alle niveaus die bouwen en implementeren van veilige Azure-oplossingen.
