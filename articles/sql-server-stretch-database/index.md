#<a name="sql-server-stretch-database-documentation"></a>Documentatie over SQL Server Stretch Database

SQL Server Stretch Database migreert uw koude gegevens transparant en veilig naar de Microsoft Azure-cloud. SQL Server Stretch Database maakt gegevens die u niet regelmatig gebruikt, op kostenefficiÃ«nte manier beschikbaar. U hoeft geen wijzigingen aan te brengen in uw query's of toepassingen en uw gegevens zijn ook tijdens de migratie beveiligd. Stretch Database is bedoeld voor transactionele databases met een grote hoeveelheid gegevens die niet regelmatig worden gebruikt en die doorgaans in een klein aantal tabellen zijn opgeslagen.
