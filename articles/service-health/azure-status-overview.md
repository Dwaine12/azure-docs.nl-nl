# <a name="azure-status"></a>Status van Azure
[Status van Azure](https://azure.microsoft.com/status/) biedt u een algemeen overzicht van de status van Azure-services. Met de status van Azure krijgt u actuele informatie over de servicebeschikbaarheid van de. Iedereen heeft toegang tot Azure-status en kan weergave alle services die rapporteren over de status van de service. 

![Pagina Azure status](./media/azure-status-overview/azure-status.PNG)

## <a name="azure-status-updates"></a>Status van Azure-updates
De pagina Azure status wordt in realtime bijgewerkt als de status van Azure-services wijzigen. Als u de pagina Azure status open laat, kunt u de snelheid waarmee de pagina wordt vernieuwd met nieuwe gegevens kunt beheren. Bovenaan ziet u de laatste keer dat de pagina is bijgewerkt.

![Status van Azure vernieuwen](./media/azure-status-overview/update.PNG)

## <a name="azure-status-history"></a>Statusgeschiedenis van Azure
Hoewel op de pagina Azure status altijd de meest recente statusgegevens, vindt u de oudere gebeurtenissen met behulp van de [geschiedenis-pagina Azure status](https://azure.microsoft.com/status/history/).

U kunt er gebeurtenissen bekijken om tot 90 dagen in het verleden, met inbegrip van een voorlopige hoofdoorzaak te achterhalen, voor risicobeperking en de volgende stappen voor Azure.

## <a name="rss-feed"></a>RSS-Feed
Status van Azure biedt ook [een RSS-feed](https://azure.microsoft.com/en-us/status/feed/) van wijzigingen in de status van Azure-services die u kunt zich abonneren.
