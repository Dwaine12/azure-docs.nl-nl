# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Richtlijnen voor Azure Data Lake Storage Gen1 klanten

Azure Data Lake Storage Gen2 Preview biedt systeemeigen Blob storage functies, waaronder gegevens in lagen, levenscyclusbeheer en verbeterde netwerkbeveiliging, naast het beschikbaar in Data Lake Storage Gen1 compatibel is met Hadoop-bestandssysteem.

Als u een bestaande Data Lake Storage Gen1-klant bent, Microsoft raadt u aan alle nieuwe en verbeterde mogelijkheden van Data Lake Storage Gen2 evalueren en feedback geven door [die deel uitmaken van de Preview-versie](https://aka.ms/adlsgen2signup). Data Lake Storage Gen1 blijft ook worden ondersteund. Data Lake Storage Gen1 en Gen2 kunnen naast elkaar bestaan, zodat uw bestaande bronnen en toepassingen die gebruikmaken van Data Lake Storage Gen1 blijven werken zonder updates. Wanneer het u uitkomt, kunt u een migratieplan voor Data Lake Storage Gen2 ontwikkelen.

Als u Data Lake Storage Gen1 momenteel evalueren wilt, raadt Microsoft u deelnemen aan de preview Gen2 en evalueren van de nieuwe en verbeterde mogelijkheden die door Data Lake Storage Gen2. Op dezelfde manier als u een bestaande Blob storage-klant bent, raadt Microsoft u aan te evalueren of een compatibel is met Hadoop-bestandssysteem geschikt is voor de behoeften van uw analyse is.
