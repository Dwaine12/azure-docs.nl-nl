# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Richtlijnen voor Azure Data Lake Storage Gen1 klanten

Azure Data Lake Storage Gen2 Preview biedt systeemeigen Blob storage functionaliteit, waaronder gegevens lagen levenscyclusbeheer en verbeterde netwerkbeveiliging, naast het compatibel is met Hadoop-bestandssysteem beschikbaar is in Data Lake Storage Gen1.

Als u een bestaande Data Lake Storage Gen1-klant bent, Microsoft raadt u alle nieuwe en verbeterde mogelijkheden van Data Lake Storage Gen2 evalueren en feedback door [die deel uitmaken van de preview](https://aka.ms/adlsgen2signup). Data Lake Storage Gen1 blijven ook worden ondersteund. Data Lake Storage Gen1 en Gen2 kan worden gecombineerd, zodat uw bestaande bronnen en toepassingen die gebruikmaken van de Data Lake Storage Gen1 blijven werken zonder updates. U kunt een migratieplan naar Data Lake Storage Gen2 ontwikkelen op uw gemak.

Als u momenteel Data Lake Storage Gen1 evalueren wilt, wordt Microsoft raadt u deelnemen aan de preview Gen2 en evalueren van de nieuwe en verbeterde mogelijkheden van Data Lake Storage Gen2 beschikbaar wordt gesteld. Op dezelfde manier als u een bestaande Blob storage-klant bent, adviseert Microsoft om te beoordelen of een compatibel is met Hadoop-bestandssysteem geschikt voor de behoeften van uw analyse is.
