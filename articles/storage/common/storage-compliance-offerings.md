# <a name="azure-storage-compliance-offerings"></a>Azure Storage compliance-aanbiedingen

Waarmee organisaties kunnen voldoen aan biedt nationale, regionale en branchespecifieke voorwaarden die voor het verzamelen en gebruiken van personen gegevens, Microsoft Azure en Azure Storage de meest uitgebreide set certificeringen en verklaringen van een cloudserviceprovider.

U vindt hieronder compliance-aanbiedingen in Azure Storage om te controleren of uw service met behulp van Azure Storage-service heeft vastgesteld. Ze zijn van toepassing op de volgende aanbiedingen voor Azure Storage: Blobs, bestanden, wachtrijen, tabellen, schijven, Cool Storage en Premium-opslag.
