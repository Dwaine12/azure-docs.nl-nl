# <a name="get-help-from-machine-learning-live-chat-support---deprecated"></a>Hulp bij Machine Learning Live chatten Support - AFGESCHAFT

Niet meer beschikbaar.

[Azure Machine Learning Studio](what-is-ml-studio.md) biedt een intuïtieve interface voor het bouwen van machine learning-modellen. Er is een [galerie](gallery-how-to-use-contribute-publish.md) van voorbeelden die is bijgedragen door de community kunt u aan de slag. En er is een [Forum](https://social.msdn.microsoft.com/forums/azure/home?forum=MachineLearning) waar u kunt vragen om te voorkomen dat u gaat. 

Maar soms moet u een snelle vraag om de blokkering van u te vragen. In [Machine Learning Studio](http://studio.azureml.net/Home) Zoek het chat-pictogram in de bovenste navigatiebalk.  Als u dit pictogram ziet, betekent dit dat lid is van het productteam om u te helpen in realtime online is.

![Azure ML Chat](./media/live-chat/AzureMLChatNavBar.png)

Typ uw vraag en uw antwoorden!

![Azure ML Chat dialoogvenster](./media/live-chat/AzureMLChat.png)

**Zie het pictogram Live chatten niet?**

De Live chatten personeelsbezetting door leden van de Machine Learning-team. Als er geen het pictogram live chatten is omdat het team momenteel niet beschikbaar is. Het kan bijvoorbeeld zijn buiten de normale werkuren. 

U kunt uw vragen altijd boeken op de [Machine Learning-Forum](https://social.msdn.microsoft.com/forums/azure/home?forum=MachineLearning), waar ze worden gezien door leden van de Machine Learning-team en andere geïnformeerde gegevenswetenschappers en ontwikkelaars.
