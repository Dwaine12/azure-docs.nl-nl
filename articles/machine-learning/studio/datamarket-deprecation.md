# <a name="datamarket-is-now-deprecated"></a>DataMarket is nu verouderd.

De [Microsoft DataMarket](https://datamarket.azure.com/datasets) is nu afgeschaft en dus de Machine Learning-API's die zijn gehost er nu ook zijn afgeschaft.

U kunt deze mogelijkheden (en meer) nu zoeken de [cognitieve API's voor Services](https://www.microsoft.com/cognitive-services).

## <a name="text-analytics-api"></a>Tekstanalyse API

Vorige DataMarket Text Analytics API klanten moeten beginnen met de [cognitieve Services Text Analytics API](https://www.microsoft.com/cognitive-services/text-analytics-api).
De cognitieve Services Text Analytics API ondersteunt alle scenario's die de oudere versie van de DataMarket hebt gedaan, evenals aanvullende scenario's zoals taal wordt gedetecteerd en onderwerp detectie.
