## <a name="get-support-for-azure-machine-learning-services"></a>Ondersteuning krijgen voor Azure Machine Learning-services

U kunt ondersteuning voor Azure Machine Learning-services in de volgende manieren:

+ **Technische ondersteuning**: Azure Neem contact op met technische ondersteuning via bezoeken [ondersteuning van Azure](https://azure.microsoft.com/support/options/) en Machine Learning te selecteren. 

+ **MSDN-gebruikersforum**: vragen, beantwoorden van vragen en verbinding maken met andere gebruikers in de [Azure Machine Learning serviceondersteuning forum op MSDN](https://social.msdn.microsoft.com/Forums/windowsdesktop/home?forum=MachineLearning).

+ **Stack-overloop**: Ga naar de Azure Machine Learning-community op [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning) gemarkeerd met 'Azure Machine Learning'.

+ **Delen van suggesties** en functies van aanvragen in onze [Azure Machine Learning Feedback kanaal](https://feedback.azure.com/forums/257792-machine-learning), dit is ook toegankelijk via de koppeling onder aan elk artikel.
