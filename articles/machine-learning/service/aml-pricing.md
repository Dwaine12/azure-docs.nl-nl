# <a name="pricing-and-billing-for-azure-machine-learning-services"></a>Prijzen en facturering voor Azure Machine Learning-Services

Voor de volledige informatie of om een factuur voorbeeld weer te geven, gaat u naar de [Azure pagina met prijzen](https://azure.microsoft.com/pricing/details/machine-learning-services/).  

Hier volgen enkele veelgestelde vragen over facturering en prijzen.

## <a name="can-i-use-azure-machine-learning-for-free-during-preview"></a>Kan ik gebruiken Azure Machine Learning gratis tijdens de preview?    

De toepassing Azure Machine Learning-Workbench is beschikbaar gratis Azure-abonnees.

De Service experimenteren en het Modelbeheer bieden gratis lagen, naast betaald lagen, beschikbaar op een korting tijdens de openbare Preview.

## <a name="am-i-charged-based-on-how-many-experiments-i-run"></a>Ben ik in rekening gebracht op basis van hoeveel experimenten ik uitvoeren?

Nee, u kunt met de Experimenten-service net zoveel experimenten uitvoeren als u wilt. De kosten zijn uitsluitend gebaseerd op het aantal gebruikers. Rekenbronnen voor experimenten worden afzonderlijk in rekening gebracht.  We raden u aan het uitvoeren van meerdere experimenten voor het beste model voor uw oplossing aankomen. 

## <a name="am-i-charged-based-on-how-many-times-my-web-services-is-called"></a>Ben ik in rekening gebracht op basis van het aantal keren dat mijn web-services wordt aangeroepen?

Nee. Web-services kunnen zo vaak als nodig zijn, zonder gevolgen facturering Model Management worden aangeroepen. U hebt de volledige controle om uw implementaties te schalen om te voldoen aan de vereisten van uw toepassingen.

## <a name="how-can-i-scale-the-units-i-purchase-in-the-azure-machine-learning-model-management"></a>Hoe kan ik de eenheden die ik in Azure Machine Learning-Model Management aanschaffen schalen?

Kunt u het aantal eenheden, omhoog of omlaag, met behulp van de Azure-portal of de CLI. 

## <a name="what-does-a-bill-look-like"></a>Wat een factuur eruit?

Facturen worden dagelijks geproduceerd. Voor facturatiedoeleinden begint een dag om middernacht (UTC). Facturen worden maandelijks gegenereerd. Afzonderlijke kosten verbonden zijn voor Azure-services die worden gebruikt in combinatie met Azure Machine Learning. Kosten kunnen bevatten, maar zijn niet beperkt tot: 
- COMPUTE-kosten
- HDInsight
- Azure Container Service
- Azure Container Registry 
- Azure Blob Storage
- Application Insights
- Azure Key Vault
- Visual Studio Team Services
- Azure Event Hub
- Azure Stream Analytics voor meer informatie of om weer te geven van een factuur voorbeeld gaat u naar de [Azure pagina met prijzen](https://azure.microsoft.com/pricing/details/machine-learning-services/).
