# <a name="python-packages-for-azure-machine-learning"></a>Python-pakketten voor Azure Machine Learning

Meer informatie over de eigen Python-pakketten van Microsoft voor Azure Machine Learning. U kunt deze bibliotheken en -functies gebruiken in combinatie met andere pakketten open source of van derden, maar de eigen uw code moet worden uitgevoerd op basis van een service of op een computer waarmee de interpreters Python-pakketten gebruiken.

De Azure Machine Learning-pakketten zijn **Python-pip installeerbare extensies voor Azure Machine Learning** waarmee gegevenswetenschappers en AI-ontwikkelaars om snel te bouwen en implementeren uiterst nauwkeurige machine learning- en deep learning modellen voor verschillende domeinen.

<a name="amlpcv"></a>
## <a name="azure-ml-package-for-computer-vision"></a>Azure ML-pakket voor Computer Vision

U kunt met Azure ML-pakket voor Computer Vision, bouwen, afstemmen en implementeren van deep learning-modellen voor classificatie van afbeeldingen, object-detectie en gelijkenis afbeelding.

Probeer de volgende stappen voor dit pakket:
1. [Download](https://aka.ms/aml-packages/vision/download) het pakket.
1. Lees de [installeren docs](https://aka.ms/aml-packages/vision).
1. [Bouwen en implementeren van een computer vision-model](how-to-build-deploy-image-classification-models.md) met een Jupyter-notebook.
1. Verken het pakket [referentiedocumentatie](https://aka.ms/aml-packages/vision).
