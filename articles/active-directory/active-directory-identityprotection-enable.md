# <a name="enabling-azure-active-directory-identity-protection"></a>Inschakelen van beveiliging voor Azure Active Directory-identiteit
Azure Active Directory: Identity Protection is een nieuwe mogelijkheid biedt een geconsolideerde weergave verdachte activiteiten voor aanmelden en mogelijke beveiligingsproblemen en kunt u beschermen met meldingen, herstel aanbevelingen en beleid op basis van risico uw bedrijf. 

De service detecteert verdachte activiteiten voor eindgebruikers en identiteiten bevoegdheden (admin) op basis van signalen zoals beveiligingsaanvallen, gelekte referenties, aanmeldingen vanaf onbekende locaties, ge√Ønfecteerde apparaten als bescherming tegen deze activiteiten in realtime. Belangrijker is, op basis van deze verdachte activiteiten, de ernst van een gebruiker risico wordt berekend en beleid op basis van risico's kunnen worden geconfigureerd en automatisch de identiteiten van uw organisatie te beveiligen. Zie voor meer informatie [Azure Active Directory: Identity Protection](active-directory-identityprotection.md).

In dit onderwerp wordt beschreven hoe Azure Active Directory identiteitsbeveiliging in te schakelen.

## <a name="steps-to-enable-azure-active-directory-identity-protection"></a>Stappen voor het inschakelen van Azure Active Directory: Identity Protection
1. [Eenmalige aanmelding](https://ms.portal.azure.com/) naar uw Azure-portal als globale beheerder. 
2. Klik in de Azure-portal op **Marketplace**.
   
    ![Maak](./media/active-directory-identityprotection-enable/01.png "maken")
3. Klik in de lijst met toepassingen op **beveiliging en identiteit**.
   
    ![Maak](./media/active-directory-identityprotection-enable/02.png "maken")
4. Klik op **Azure AD Identity Protection**.
   
    ![Maak](./media/active-directory-identityprotection-enable/03.png "maken")
5. Op de **Azure AD Identity Protection** blade, klikt u op **maken**.
   
    ![Maak](./media/active-directory-identityprotection-enable/04.png "maken")
