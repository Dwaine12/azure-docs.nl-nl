# <a name="microsoft-authenticator-app"></a>Microsoft Authenticator-app

>[!Important]
>Deze inhoud is bedoeld voor gebruikers. Als u een beheerder bent, kunt u meer informatie vinden over hoe u uw Azure Active Directory-omgeving (Azure AD) instelt en beheert in de [documentatie voor Azure Active Directory](https://docs.microsoft.com/en-us/azure/active-directory).

Met de Microsoft Authenticator-app kunt u uw accounts beveiligen met behulp van verificatie in twee stappen nadat u het wachtwoord hebt ingevoerd. Maar wist u ook dat dit het wachtwoord voor uw persoonlijke Microsoft-account kan vervangen? Meer informatie over onder andere het instellen en gebruiken van de Microsoft Authenticator-app.
