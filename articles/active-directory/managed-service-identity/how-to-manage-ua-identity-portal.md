# <a name="create-list-or-delete-a-user-assigned-managed-service-identity-msi-using-the-azure-portal"></a>Maken, weergeven of verwijderen van een gebruiker toegewezen Managed Service Identity (MSI) met behulp van de Azure portal

[!INCLUDE[preview-notice](~/includes/active-directory-msi-preview-notice-ua.md)]

Beheerde Service-identiteit biedt Azure-services met een beheerde identiteit in Azure Active Directory. U kunt deze identiteit gebruiken voor verificatie bij services die ondersteuning bieden voor Azure AD-verificatie, zonder de referenties in uw code. 

Op dit moment is het niet mogelijk te maken, weergeven of verwijderen van een gebruiker toegewezen beheerde identiteit met Azure portal.  Kijk binnenkort voor updates.
