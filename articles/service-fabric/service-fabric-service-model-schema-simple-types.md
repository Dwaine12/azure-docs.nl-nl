# <a name="service-model-xml-schema-simple-types"></a>Service model XML-schema eenvoudige typen

## <a name="fabricuri-simpletype"></a>FabricUri simpleType-element
Een URI die wordt gebruikt als een stabiele id van services door Microsoft Azure Service Fabric Naming-systeem. 

### <a name="xml-source"></a>XML-bron
```xml
<xs:simpleType xmlns:xs="http://www.w3.org/2001/XMLSchema" xmlns="http://schemas.microsoft.com/2011/01/fabric" name="FabricUri">
    <xs:annotation>
      <xs:documentation>A URI that is used as a stable identifier of services by Microsoft Azure Service Fabric Naming system. </xs:documentation>
    </xs:annotation>
    <xs:restriction base="xs:anyURI"/>
  </xs:simpleType>
  
```
