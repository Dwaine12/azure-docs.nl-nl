# <a name="supported-postgresql-database-versions"></a>Versies van de ondersteunde PostgreSQL-Database
Microsoft is erop gericht ter ondersteuning van n-2 versies van de PostgreSQL-engine in de Azure-Database voor PostgreSQL-service, wat betekent dat de primaire momenteel uitgebrachte versie (n) en de twee vorige primaire versies-(2).

Azure PostgreSQL-Database ondersteunt momenteel de volgende versies:

## <a name="postgresql-version-103"></a>PostgreSQL versie 10.3
Raadpleeg de [PostgreSQL documentatie](https://www.postgresql.org/docs/10/static/release-10-3.html) voor meer informatie over verbeteringen en verbeteringen in PostgreSQL 10.3.

## <a name="postgresql-version-967"></a>PostgreSQL versie 9.6.7
Raadpleeg de [PostgreSQL documentatie](https://www.postgresql.org/docs/9.6/static/release-9-6-7.html) voor meer informatie over verbeteringen en verbeteringen in PostgreSQL 9.6.7.

## <a name="postgresql-version-9511"></a>PostgreSQL versie 9.5.11
Raadpleeg de [PostgreSQL documentatie](https://www.postgresql.org/docs/9.5/static/release-9-5-11.html) voor meer informatie over verbeteringen en oplossingen in PostgreSQL 9.5.11.

## <a name="managing-updates-and-upgrades"></a>Het beheer van updates en upgrades
Azure-Database voor PostgreSQL beheert automatisch patchen voor secundaire versie-updates. Primaire versie upgrade wordt momenteel niet ondersteund. Bijvoorbeeld, wordt een upgrade van PostgreSQL 9.5 naar PostgreSQL 9,6 niet ondersteund.
