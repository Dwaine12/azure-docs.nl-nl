#<a name="sql-database-documentation"></a>Documentatie voor SQL Database

Azure SQL Database is een relationele database-service in de cloud met missiekritische mogelijkheden gebaseerd op de Microsoft SQL Server-engine. SQL Database biedt voorspelbare prestaties, schaalbaarheid zonder uitvaltijd, bedrijfscontinu√Øteit en gegevensbeveiliging - en dat allemaal vrijwel zonder beheer. Omdat SQL Database is gebaseerd op de SQL Server-engine, worden bestaande SQL Server-hulpprogramma's, -bibliotheken en -API's ondersteund voor een soepele verplaatsing en uitbreiding naar de cloud.
