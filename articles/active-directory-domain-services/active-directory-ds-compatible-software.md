## <a name="one-identity-authentication-services"></a>Verificatieservices één identiteit

![Één identiteit](./media/compatible-software-icons/OneIdentity.png)

Een identiteit verificatieservices kunnen Unix, Linux en Mac OS X-systemen om de toegang, verificatie en autorisatie van een organisatie bestaande Active Directory (AD)-infrastructuur gebruiken. Verificatieservices biedt nu ondersteuning voor Azure Active Directory Domain Services inschakelen niet-Windows-bronnen te kunnen gebruiken hetzelfde next generation-platform die al gebruikmaken van uw bestaande SaaS-oplossingen.
