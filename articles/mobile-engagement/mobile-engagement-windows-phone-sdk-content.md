Dit document bevat en beschrijft de inhoud die is geïmplementeerd door de SDK in uw toepassing.

## <a name="the-resources-folder"></a>De `/Resources` map
Deze map bevat alle resources die behoeften van de Mobile Engagement. U kunt deze aanpassen aan uw app ook aanpassen.

* `EngagementIconNotification.png` : Het merk-pictogram weergegeven aan de linkerkant van een melding vervangen deze door uw merk-pictogram.
* `EngagementIconOk.png` : ApplicationBar-pagina's het 'Ok' pictogram van het bereik.
* `EngagementIconCancel.png` : ApplicationBar-pagina's het pictogram 'Annuleren' van het bereik.
* `EngagementIconCloseLight.png` : De pictogram "Sluiten" van de Mobile Engagement bereiken melding voor de Windows Phone licht thema.
* `EngagementIconCloseDark.png` : De pictogram "Sluiten" van de Mobile Engagement bereiken melding voor de Windows Phone donkere thema.
* `EngagementConfiguration.xml` : De Mobile Engagement-configuratiebestand, dit is waar u kunt Engagement instellingen aanpassen (Engagement verbindingsreeks, rapport vastloopt enzovoort).
