# <a name="pricing"></a>Prijzen

## <a name="compute-cost"></a>COMPUTE-kosten
Azure Cloud-Shell op een machine gratis geleverd door Azure wordt uitgevoerd, maar vereist een Azure-bestandsshare te gebruiken. 

## <a name="storage-cost"></a>Opslagkosten
Cloud-Shell is een nieuwe of bestaande bestandsshare in Azure worden gekoppeld om te blijven behouden bestanden over de sessies vereist. Opslag geregelde kosten in rekening worden gebracht.

Controleer [hier voor meer informatie over Azure Files kosten](https://azure.microsoft.com/pricing/details/storage/files/).
