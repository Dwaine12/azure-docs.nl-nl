# <a name="natural-language-support-for-azure-cognitive-services"></a>Ondersteuning van natuurlijke taal voor Azure Cognitive Services

Azure Cognitive Services kunt u toepassingen ontwikkelen die zien, horen, spreken met en inzicht krijgen in uw gebruikers. Tussen deze services, worden meer dan drie dozijn talen ondersteund, wordt gebruikers om te communiceren met uw toepassing op een natuurlijke manier.

In dit artikel is onderverdeeld in twee secties. De eerste geeft een lijst van core talen worden ondersteund voor Azure Cognitive Services. Het tweede gedeelte richt zich op aanvullende taalondersteuning voor elk van de services. Taal beschikbaarheid kan variëren per land/regio en markt.

## <a name="core-languages"></a>Belangrijkste talen

Deze core-talen worden ondersteund voor Azure Cognitive Services:

* Chinese¹
* Engels
* Frans
* Duits
* Italiaans
* Japans
* Korean²
* Portugees
* Spaans

> [!NOTE]
> ¹ text Analytics wordt niet ondersteund. <br/>
> ² LUIS, Speech Service: spraak-naar-tekst en Video Indexer: spraak-naar-tekst worden niet ondersteund.

## <a name="additional-language-availability-by-service"></a>Beschikbaarheid van extra taal per service

Deze tabellen markeren taal beschikbaarheid per servicecategorie. Core-talen die zijn uitgesloten. Gebruik de onderstaande koppelingen voor meer informatie over ondersteuning voor andere talen en land/regio en markt beschikbaarheid.
