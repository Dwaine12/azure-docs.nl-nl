# <a name="entity-search-endpoints"></a>Entiteit zoeken eindpunten
De **entiteit Search API** bevat één eindpunt.

## <a name="endpoint"></a>Eindpunt
Als de zoekresultaten entiteit, een aanvraag te verzenden naar het volgende eindpunt. De kop- en URL-parameters gebruiken voor het definiëren van verdere specificaties.

Eindpunt `GET`: 
``` 
https://api.cognitive.microsoft.com/bing/v7.0/entities
```

De volgende URL-parameters zijn vereist:
- Mkt. De markt waar de resultaten afkomstig zijn uit. 
- q. De zoekopdracht entiteit.
