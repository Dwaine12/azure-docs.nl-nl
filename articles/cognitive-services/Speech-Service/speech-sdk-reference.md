# <a name="speech-sdk-reference"></a>Spraak SDK-naslaginformatie

De spraak Software Development Kit (SDK) biedt uw toepassingen toegang tot de functies van de service spraak, waardoor het gemakkelijker om spraakherkenning ingeschakeld software te ontwikkelen. Op dit moment is de SDK's bieden toegang tot **spraak naar tekst** en **spraak vertaling**. [REST] de veiligheidsmaatregelen voor Text to Speech (https://docs.microsoft.com/azure/cognitive-services/speech-service/rest-apis) aanroepen via HTTP POST.

De volgende tabel beschrijft de besturingssystemen en programmeertalen worden momenteel ondersteund.

|Programmeertaal|Platform|API-naslaginformatie|
|-|-|-|
|C#|Windows|[Bladeren](https://aka.ms/csspeech/csharpref)
|C/C++|Windows, Linux|[Bladeren](https://aka.ms/csspeech/cppref)
|Java\*|Apparaten|[Bladeren](https://aka.ms/csspeech/javaref)

\* *De Java SDK maakt deel uit van de [spraak apparaten SDK](speech-devices-sdk.md).*
