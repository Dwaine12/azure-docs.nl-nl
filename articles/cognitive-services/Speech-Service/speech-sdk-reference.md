# <a name="speech-sdk-api-reference"></a>Spraak SDK API-referentie

Spraak Software Development Kit (SDK) biedt uw toepassingen toegang tot de functies van de spraak-service, waardoor het gemakkelijker wordt om spraak ingeschakelde software te ontwikkelen. Op dit moment de SDK's bieden toegang tot **spraak naar tekst**, **Spraakomzetting**, en **bedoeling erkenning**. Een algemeen overzicht over de mogelijkheden en ondersteunde platforms kunt u vinden op de documentatie [vermelding pagina](http://aka.ms/csspeech).
