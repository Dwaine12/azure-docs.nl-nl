# <a name="upload-images"></a>Afbeeldingen uploaden #

Als u wilt de gebruikersinterface verkennen, maximaal vijf afbeeldingen te uploaden tegelijk elke installatiekopie mag maximaal 2 MB bestandsgrootte. Zorg ervoor dat uw bestandsgrootten controleren voordat u uploadt.

![Afbeeldingen uploaden](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>Verzenden voor de controle ##
Als u uiteindelijk uploaden van meer dan vijf installatiekopieën op een tijdstip, gebruikt u het pictogram Prullenbak ongewenste afbeeldingen verwijderen. Dien vervolgens de installatiekopieën voor geautomatiseerde toezicht.

![Afbeeldingen verzenden](images/1-try-image-2.png)
