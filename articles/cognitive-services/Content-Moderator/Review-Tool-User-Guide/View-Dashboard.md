# <a name="view-dashboard"></a>Bekijk het dashboard #

Klik op de **Dashboard** tabblad voor een overzicht van de belangrijkste metrische gegevens voor alle inhoud controleren in het hulpprogramma wordt uitgevoerd. Totaal aantal voltooide en in uitvoering beoordelingen voor tekst en afbeeldingen, kunt u zien. U kunt metrische gegevens zien voor uw standaardteam en voor uw subteams.
