# <a name="recommendations-api-discontinued"></a>Aanbevelingen API (afgeschaft)

De evaluatieversie aanbevelingen API is niet langer in active ontwikkeling en officieel op 15 februari is ingetrokken. 

Voor een alternatief kunt u de nieuwe open-source [aanbevelingen oplossingssjabloon](http://aka.ms/recopcs), waarmee u een engine aanbevelingen binnen uw eigen Azure-abonnement te hosten. Het ondersteunt de belangrijkste scenario's van de API aanbevelingen ook zodat u de vrijheid om te implementeren op Azure-regio en meer nauwlettend in de rekenkundige bronnen gebruikt voor het trainen van modellen en aanbevelingen fungeren. 

Meer informatie over de aanbevelingen oplossingssjabloon door te lezen [de documentatie](https://github.com/Microsoft/Product-Recommendations).
