# <a name="upgrading-code-to-the-latest-platform"></a>Code upgraden naar de meest recente platform

In dit artikel wordt uitgelegd hoe u uw Microsoft Dynamics 365 voor bewerkingen platformversie upgraden naar de meest recente versie van de platform.

## <a name="overview"></a>Overzicht

Het Microsoft Dynamics 365 voor bewerkingen platform bestaat uit de volgende onderdelen:

Dynamics 365 voor bewerkingen platform binaire bestanden, zoals Server (AOS), data management framework, de rapportage en business intelligence (BI) framework, ontwikkelprogramma's en analytics-services. De volgende structuur (AOT)-pakketten:

1. Application-platform
2. Toepassing Foundation
3. Essentials testen

**Belangrijke**: als u wilt verplaatsen naar de meest recente Dynamics 365 voor bewerkingen platform, uw Dynamics 365 voor uitvoering van bewerkingen aanpassingen (overlayering) van een van de AOT-pakketten die deel uitmaken van het platform kan niet hebben. Als u uitvoert op een platform dat ouder is dan platform 3, raadpleegt u de upgrade naar platform update 3 van een eerdere sectie build aan het einde van dit artikel.
