# <a name="create-an-azure-application-offer"></a>De aanbieding van een Azure-toepassing maken

In dit artikel wordt beschreven hoe u maken en publiceren van een vermelding van de aanbieding Azure-toepassing voor de Azure Marketplace. Elke aanbieding wordt weergegeven als een eigen entiteit in Azure Marketplace en is gekoppeld aan een of meer SKU's. Een aanbieding bestaat uit de volgende groeperingen van assets en ondersteunende services:

## <a name="new-offer-form"></a>Nieuwe aanbieding formulier

Aanmelden bij de [Cloud Partner-Portal](http://cloudpartner.azure.com/), en selecteer vervolgens **+ nieuwe aanbieding** in het menu links. Bieden op het pictogram Nieuw menu, selecteert u **Azure-toepassingen** om weer te geven de **nieuwe aanbieding** vormen en start het proces van het definiëren van assets voor een nieuwe aanbieding.

![Nieuwe aanbieding formulier](./media/azure-new-managedapp-offer.png)
