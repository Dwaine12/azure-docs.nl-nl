# <a name="billing-options"></a>Opties voor facturering  

## <a name="commercial-considerations-on-the-marketplace"></a>Commerciële overwegingen op marketplace  
De marketplace deelt niet omzet voor de volgende typen van de aanbieding.  
*   Lijst  
*   Proefversie  
*   Transact met behulp van het facturering model Bring Your Own License (BYOL)

U bent geen gefactureerd extra kosten voor uw deelname aan het winkelobjecten op marketplace.  
*   Voor meer informatie gaat u naar de Azure Marketplace deelname beleidsregels pagina zich bevindt op [azure.microsoft.com/support/legal/marketplace/participation-policies](https://azure.microsoft.com/support/legal/marketplace/participation-policies).  

## <a name="pay-as-you-go-and-bring-your-own-license-billing-options"></a>Betalen naar gebruik en breng uw eigen licentie facturering-opties  
Wanneer u een betalen naar gebruik factureringsmodel als publicatieoptie, gedeeld uw op gebruik gebaseerde software licensing omzet is 80-20 tussen u en Microsoft. Één aanbieding kan worden berekend met behulp van zowel betalen naar gebruik als BYOL factureringsmodellen. De twee factureringsmodellen naast elkaar bestaan op het niveau van de aanbieding als afzonderlijke SKU's. U kunt de factureringsmodellen configureren in uw aanbieding op Cloud Partner-Portal. 

Houd rekening met de volgende voorbeelden.  
*   Als u de optie betalen naar gebruik inschakelt, hebt u het volgende resultaat.  
    | Uw licentie kosten | USD 1,00 per uur |  
    |:--- |:--- |  
    | Azure-gebruikskosten (1-D1-Core) | $0.14 per uur |  
    | *De klant wordt gefactureerd door Microsoft* | *$1.14 per uur* |  

    In dit scenario stuklijsten Microsoft $1.14 per uur voor het gebruik van uw gepubliceerde VM-installatiekopie.  
    | Microsoft facturen | $1.14 per uur |  
    |:--- |:--- |  
    | Microsoft betaalt u 80% van uw licentie kosten | $0,80 per uur |  
    | Microsoft houdt 20% van uw licentie kosten | $0,20 per uur |  
    | Microsoft houdt de gebruikskosten van Azure | $0.14 per uur |  

*   Als u de optie BYOL inschakelt, hebt u het volgende resultaat.  
    | Uw licentie kosten | Licentiekosten onderhandeld en door u in rekening gebracht |  
    |:--- |:--- |  
    | Azure-gebruikskosten (1-D1-Core) | $0.14 per uur |  
    | *De klant wordt gefactureerd door Microsoft* | *$0.14 per uur* |  

    In dit scenario stuklijsten Microsoft $0.14 per uur voor het gebruik van uw gepubliceerde VM-installatiekopie.  
    | Microsoft facturen | $0.14 per uur |  
    |:--- |:--- |  
    | Microsoft houdt de gebruikskosten van Azure | $0.14 per uur |  
    | Microsoft houdt 0% van de kosten van uw licentie | 0,00 $ per uur |  

## <a name="single-billing-and-payment-methods"></a>Één facturering en betalingswijzen  
Een belangrijk voordeel van het gebruik van de aanbieding optie Transact is dat uw licentiekosten en Azure gebruik één gefactureerd rechtstreeks naar de klant.  

In het scenario Microsoft af- en verzamelt namens jou. Microsoft facturering verwijdert de vereiste u uw eigen inkoop relatie maken met uw klant. Één facturering kan bespaart u tijd en hulpmiddelen. Één facturering kunt u zich kunt richten op de verkoop aanvoer in plaats van de factuur verzamelen. 

## <a name="enterprise-agreement"></a>Enterprise Agreement  
Als u een klant Enterprise Agreement (EA) met Microsoft, kunt u uw EA gebruiken om te betalen voor Microsoft-producten. U kunt producten, inclusief het gebruik van uw Azure factureren. Met behulp van uw EA om te betalen is ontworpen voor organisaties die willen licentie software en in de cloud services gedurende drie of meer jaar. U verspreid betalingen in plaats van een vooraf betaling. Als u betalen naar gebruik gebruiken transact vermeldingstype, klikt u vervolgens de facturering voor uw software licentiekosten volgt de driemaandelijkse EA overschrijding factureringscyclus.  

### <a name="monetary-commitment"></a>Monetaire toezegging  
Als u een EA-klant bent, kunt u Azure u de overeenkomst toevoegen. U toevoegen Azure u de overeenkomst door het maken van een maandbedrag naar Azure. Uw maandbedrag wordt gedurende het jaar verbruikt. Uw betrokkenheid bevat een combinatie van informatie over het gebruik van de grote verscheidenheid aan services in Azure.
