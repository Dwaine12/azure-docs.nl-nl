## <a name="visual-studio-code-extension-for-ansible"></a>Visual Studio Code-uitbreiding voor Ansible

De [Visual Studio Code-uitbreiding voor Ansible](https://marketplace.visualstudio.com/items?itemName=vscoss.vscode-ansible) biedt verschillende functies voor het gebruik van Ansible van Visual Studio Code:

- Automatische voltooiing van Ansible richtlijnen, modules en invoegtoepassingen in de documentatie Ansible terwijl u typt.
- Code codefragmenten weergegeven wanneer u op &lt;Ctrl >&lt;ruimte > tijdens het ontwerpen van uw Ansible hulpmiddelen marketing en verkoop.
- Syntaxismarkering, geeft de code van uw Ansible playbook weer in verschillende kleuren en lettertypen in overeenstemming met YAML-syntaxis.
- Ansible hulpmiddelen marketing en verkoop kunnen worden uitgevoerd vanuit het venster Visual Studio Code Terminal.
    - (Alleen Windows) Ansible kan worden uitgevoerd vanaf een Docker-container.
    - (Linux en Mac OS) Ansible kan worden uitgevoerd vanuit een Docker-container of vanuit een lokale Ansible-installatie. 
- Ansible hulpmiddelen marketing en verkoop kunnen worden uitgevoerd vanuit Azure Cloud-Shell.
