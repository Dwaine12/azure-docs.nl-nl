# <a name="advisor-security-recommendations"></a>Advisor-aanbevelingen voor beveiliging

Azure Advisor biedt u een consistente, geconsolideerde weergave van de aanbevelingen voor al uw Azure-resources. Het is geïntegreerd met Azure Security Center zodat u aanbevelingen voor beveiliging. Krijgt u aanbevelingen voor beveiliging van de **Security** tabblad op de Advisor-dashboard.

Azure Security Center helpt u bij het detecteren, voorkomen van en reageren op bedreigingen dankzij een verhoogde zichtbaarheid van en controle over de beveiliging van uw Azure-resources. Er worden regelmatig de beveiligingsstatus van uw Azure-resources. Wanneer met Security Center potentiële beveiligingsproblemen worden geïdentificeerd, worden er aanbevelingen gedaan. De aanbevelingen begeleiden u bij het proces van het configureren van de besturingselementen die u nodig hebt. 

Zie voor meer informatie over aanbevelingen voor beveiliging, [aanbevelingen voor beveiliging in Azure Security Center beheren](https://azure.microsoft.com/documentation/articles/security-center-recommendations/).

## <a name="how-to-access-security-recommendations-in-azure-advisor"></a>Toegang tot de aanbevelingen voor beveiliging in Azure Advisor

1. Aanmelden bij de [Azure-portal](https://portal.azure.com), en open vervolgens [Advisor](https://aka.ms/azureadvisordashboard).

2.  Klik op de Advisor-dashboard op de **Security** tabblad.
