# Get-AzureRmFrontDoor
Configuratie van de voordeur

## SYNTAXIS

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHRIJVING
De **Get-AzureRmFrontDoor** cmdletGet haalt alle Front-deuren in het huidige abonnement die informatie komt overeen met het opgegeven.
