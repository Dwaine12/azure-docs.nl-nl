## <a name="initialize-debug-assets-with-the-vs-code-extension"></a>Foutopsporing activa met de extensie tegenover Code initialiseren
Moet u eerst het CodeProject om zo te configureren tegenover Code zullen communiceren met onze ontwikkelomgeving in Azure. De Code van de VS-extensie voor Azure Dev spaties bevat een helper-opdracht debug configuratie instellen. 

Open de **opdracht palet** (met behulp van de **weergave | Opdracht palet** menu), en gebruik automatisch aanvullen, typt en selecteert deze opdracht: `Azure Dev Spaces: Create configuration files for connected development`. 

Hiermee voegt u de configuratie van de foutopsporing voor Azure Dev spaties onder de `.vscode` map.

![](../media/common/command-palette.png)

> [!Important]
> Sluit en VS Code opnieuw openen voordat u doorgaat vanwege een fout.
