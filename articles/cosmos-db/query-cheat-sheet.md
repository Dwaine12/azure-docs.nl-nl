# <a name="azure-cosmos-db-query-cheat-sheets"></a>Azure DB Cosmos query bedriegen bladen

De **Azure Cosmos DB query bedriegen bladen** snel te schrijven query's voor uw gegevens door algemene databasequery's, bewerkingen, functies en operators in easy print PDF-verwijzingspagina's weer te geven. De bladen bedriegen bevatten naslaginformatie voor de SQL-API, MongoDB-API, tabel-API en Gremlin/Graph API. 

Kies in het downloaden van een a4-formaat of A3-formaat. 

## <a name="letter-sized-cheat-sheets"></a>Letter-formaat bedriegen bladen

Download de [Azure Cosmos DB letter formaat query bedriegen bladen](http://go.microsoft.com/fwlink/?LinkId=623215) als u wilt afdrukken op papier letter-formaat (8.5 "x 11").

![Azure DB Cosmos query bedriegen-bladen - letter-formaat, SQL-API, JavaScript, MongoDB-API, Graph API en tabel API-query's en-functies](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-letter.png)

## <a name="oversized-cheat-sheets"></a>Grote bedriegen bladen
Download de [Azure Cosmos DB A3-formaat query bedriegen bladen](https://go.microsoft.com/fwlink/?linkid=870413) als u van plan bent om af te drukken met behulp van een plotter of grootschalige printer op papier A3-formaat (11,7 "x 16,5").

![Azure DB Cosmos query bedriegen bladen - A3-formaat, SQL-API, JavaScript, MongoDB-API, Graph API en tabel API-query's en-functies](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-a3.png)
