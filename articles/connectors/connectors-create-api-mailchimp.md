# <a name="get-started-with-the-mailchimp-connector"></a>Aan de slag met de MailChimp-connector
MailChimp is een SaaS-service waarmee bedrijven beheren en automatiseren e marketing activiteiten, waaronder het verzenden van e-mailberichten marketing, geautomatiseerde berichten en gerichte campagnes.

U kunt aan de slag door het maken van een logische app nu, Zie [een logische app maken](../logic-apps/quickstart-create-first-logic-app-workflow.md).

## <a name="create-a-connection-to-mailchimp"></a>Maak een verbinding met MailChimp
Logische apps maakt met MailChimp, moet u eerst maken een **verbinding** Geef vervolgens de details voor de volgende eigenschappen:

| Eigenschap | Vereist | Beschrijving |
| --- | --- | --- |
| Token |Ja |MailChimp-referenties opgeven |
