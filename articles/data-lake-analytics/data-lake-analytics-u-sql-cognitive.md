# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Aan de slag met de herkenning van U-SQL

## <a name="overview"></a>Overzicht
Cognitieve functionaliteit voor U-SQL kunnen ontwikkelaars put intelligence gebruiken in hun big data-programma. 

De volgende cognitieve mogelijkheden zijn beschikbaar:
* Imaging: Detecteer gezichten [voorbeeld](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Imaging: Detecteren emoties [voorbeeld](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Imaging: Detecteren van objecten (tags) [voorbeeld](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Imaging: OCR (optische tekenherkenning) [voorbeeld](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Tekst: Sleutel vindt er sleuteltermextractie plaats & Sentimentanalyse [voorbeeld](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)

## <a name="registering-cognitive-extensions-in-u-sql"></a>Registreren van cognitieve-extensies in U-SQL
