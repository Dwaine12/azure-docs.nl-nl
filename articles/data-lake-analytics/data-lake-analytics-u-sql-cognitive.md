# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Aan de slag met de cognitieve mogelijkheden van U-SQL

## <a name="overview"></a>Overzicht
Cognitieve mogelijkheden voor de U-SQL kunnen ontwikkelaars put intelligence gebruiken in hun big data-programma's. 

De volgende cognitieve mogelijkheden zijn beschikbaar:
* Imaging: Detecteren vlakken [voorbeeld](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Imaging: Detecteren emotion [voorbeeld](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Imaging: Objecten (tagging) opsporen [voorbeeld](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Imaging: OCR (OCR) [voorbeeld](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Tekst: Sleutel woordgroep extractie & gevoel analyse [voorbeeld](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)

## <a name="registering-cognitive-extensions-in-u-sql"></a>Registreren van cognitieve uitbreidingen in U-SQL
