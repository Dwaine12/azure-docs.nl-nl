# <a name="china-content-delivery-with-azure-cdn"></a>China leveren van inhoud met Azure CDN

Azure inhoud Delivery Network (CDN) globale kunt inhoud leveren aan gebruikers China met punt-of-presence (POP)-locaties in de buurt China of een pop-server waarmee de beste prestaties op aanvragen die afkomstig zijn uit China. Echter, als China een belangrijke markt voor uw klanten is en ze nodig hebben snelle prestaties, overweeg het gebruik van Azure CDN China in plaats daarvan.

Azure CDN China verschilt van Azure CDN globale levert inhoud van de POP's binnen China door de samenwerking met een aantal lokale providers. Als gevolg van Chinees naleving en regelgeving, moet u een afzonderlijk abonnement voor het gebruik van Azure CDN China registreren en uw websites moeten een ICP-licentie hebt. De portal en API optreden als u wilt inschakelen en beheren van inhoud levering is vrijwel identiek tussen Azure CDN globale en Azure CDN China.

## <a name="comparison-of-azure-cdn-global-and-azure-cdn-china"></a>Vergelijking van Azure CDN globale en Azure CDN China

Azure CDN globale en Azure CDN China hebt de volgende functies:

- Azure CDN globale:

     - Portal: https://portal.azure.com  

     - Voert contentlevering buiten China

     - Vier Prijscategorieën: Microsoft standard, standard van Verizon Verizon premium en standard van Akamai

     - [Documentatie](https://docs.microsoft.com/en-us/azure/cdn/)

- Azure CDN China:

     - Portal: https://portal.azure.cn

     - Voert leveren van inhoud in China

     - Twee Prijscategorieën: Standard en premium

     - [Documentatie](https://docs.azure.cn/en-us/cdn/)
