#### <a name="expressroute-limits"></a>Limieten voor ExpressRoute
De volgende limieten gelden voor ExpressRoute-resources per abonnement.

| Resource | Standaardlimiet |
| --- | --- |
| ExpressRoute-circuits per abonnement |10 |
| ExpressRoute-circuits per regio per abonnement (Azure Resource Manager) |10 |
| Maximum aantal routes voor persoonlijke Azure-peering met ExpressRoute standard |4,000 |
| Maximum aantal routes voor persoonlijke Azure-peering met ExpressRoute premium-invoegonderdeel |10.000 |
| Maximum aantal routes voor Azure-Microsoft-peering met ExpressRoute standard |200 |
| Maximum aantal routes voor Azure-Microsoft-peering met ExpressRoute premium-invoegonderdeel |200 |
| Maximum aantal ExpressRoute-circuits die zijn gekoppeld aan hetzelfde virtuele netwerk op verschillende locaties voor peering |4 |
| Aantal virtuele netwerk-koppelingen per ExpressRoute-circuit toegestaan |Zie de onderstaande tabel |
