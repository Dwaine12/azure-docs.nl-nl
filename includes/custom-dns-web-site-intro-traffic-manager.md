Wanneer u een Microsoft Azure Traffic Manager gebruikt om verkeer te verdelen naar uw Azure-Website, die website vervolgens kan worden geopend met behulp van de  **\*. trafficmanager.net** domeinnaam toegewezen door Azure. U kunt ook een aangepaste domeinnaam, zoals www.contoso.com, koppelen aan uw website om te voorzien van een meer herkenbare domeinnaam voor uw gebruikers.
