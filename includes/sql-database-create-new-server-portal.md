## <a name="create-an-azure-sql-database-logical-server"></a>Een logische server met Azure SQL Database maken
Voer de volgende stappen uit in Azure Portal om een logische server met Azure SQL Database te maken.

1. Als u momenteel niet bent verbonden, maakt u eerst verbinding met [Azure Portal](http://portal.azure.com).
2. Klik op **Nieuw**, klik op **Gegevens en opslag** en ga naar **SQL Database**.
3. Klik op **SQL Database** om de blade SQL Database te openen.
4. Geef de waarden op voor de volgende servereigenschappen:
   
   * Servernaam
   * Aanmeldgegevens van serverbeheerder
   * Wachtwoord
   * Abonnement (alleen als u meerdere abonnementen hebt)
   * Resourcegroep (nieuwe of bestaande)
   * Locatie
5. Klik op **Create**. In het systeemvak ziet u dat de implementatie is gestart.
6. Wacht tot de implementatie is voltooid voordat u verder gaat met de volgende stap.
