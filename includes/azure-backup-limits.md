De volgende limieten gelden voor Azure Backup.

| **Limiet** | **Standaard** |
| --- | --- |
| Servers/machines dat kan worden geregistreerd in een kluis | Windows Server/Windows-Client/System Center DPM: 50 <br/><br/> IaaS-VM's: 1000  |
| Grootte van een gegevensbron in de kluis opslag |Maximaal 54400 GB. De limiet niet van toepassing op IaaS VM backup |
| Back-upkluizen in een Azure-abonnement |500 kluizen per regio |
| Dagelijkse back-ups plannen |Windows-Server/Client: 3 per dag<br/> System Center DPM: 2 per dag <br/> IaaS-VM's: Één keer per dag  |
| Gegevensschijven die zijn gekoppeld aan een Azure-VM voor back-up | 32 |
| Afzonderlijke schijf die is gekoppeld aan Azure-VM voor back-up| 4095 GB|
