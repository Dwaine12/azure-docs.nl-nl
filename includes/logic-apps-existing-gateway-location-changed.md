**Q**: Waarom is de locatie voor mijn bestaande gateway wijzigen? <br/>
**Een**: Gateway-resources die zijn gemaakt vóór 3 mei 2017 zijn verplaatst naar de oorspronkelijke locatie van de Azure AD-tenant voor het werk of school-account dat deze gateways gemaakt. Deze wijzigingen al dan niet mogen echter van invloed op momenteel wordt uitgevoerd logic apps, die moeten blijven gewoon werken. Gateway-resource-locaties kunnen met algemene beschikbaarheid van de gateway in mei verschillen van logische app locaties.
