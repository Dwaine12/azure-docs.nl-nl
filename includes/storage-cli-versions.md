## <a name="versions-of-the-azure-cli"></a>Versies van de Azure CLI

Er zijn momenteel twee versies van de Azure-opdrachtregelinterface (CLI) beschikbaar:

* [Azure CLI](../articles/storage/common/storage-azure-cli.md): de huidige versie van de Azure CLI, voor gebruik met het Resource Manager-implementatiemodel.
* [Azure CLI voor klassieke](../articles/storage/common/storage-azure-cli-nodejs.md): een oudere versie van de Azure CLI, bedoeld voor gebruik met het klassieke implementatiemodel.
